Chinese：
这是一个字体压缩包，运行在基于magisk 19.4版本之上，如果您想要在高版本的magisk上使用，您可以替换压缩包内的magisk文件，达到您想要的效果。如果发生意外情况，本人概不负责，但可以提供技术支持。不管使用任何改变系统的文件，都建议您提前做好备份工作。

English：
This is a font compression package, running on the basis of magick 19.4. If you want to use it on a higher version of magick, you can replace the magick file in the compression package to achieve the desired effect. In case of any accident, I will not be responsible, but I can provide technical support. No matter you use any files that change the system, it is recommended that you do a good backup in advance.
